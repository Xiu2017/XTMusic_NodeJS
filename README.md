# XTMusic_NodeJS
![Image text](https://raw.githubusercontent.com/Xiu2017/XTMusic_NodeJS/master/webapps/image/README/logo.png)

基于NodeJS作为后端，少量Vue作为前端，前后端分离设计的一款在线播放器。

目前在线接口支持网易云api、酷狗api和QQ音乐api

体验地址：http://193.112.183.51/
